# 🚄 Professional Hydrogen Train Analytics Dashboard

A professional **Hydrogen Train Analytics Dashboard** built using **Python, Pandas, and Matplotlib**. The dashboard analyzes hydrogen-powered train operations through KPI cards and interactive visualizations, helping monitor transportation efficiency, sustainability, passenger growth, and operational performance.

---

## 📊 Dashboard Features

- 🚆 Total Distance Covered KPI
- ⛽ Hydrogen Fuel Consumption KPI
- 🌱 CO₂ Emissions Saved KPI
- 👥 Passenger Count KPI
- 🚄 Average Train Speed KPI
- ⚡ Energy Efficiency KPI
- 📈 Monthly Distance Trend
- ⛽ Hydrogen Consumption Analysis
- 🌱 CO₂ Savings Analysis
- 👥 Passenger Growth Trend
- ⚡ Energy Efficiency Trend
- 🔧 Maintenance Cost Analysis
- 🛤️ Route Performance Analysis
- 🥧 Passenger Distribution
- 📊 Distance vs Hydrogen Consumption
- 🌙 Professional Dark Theme
